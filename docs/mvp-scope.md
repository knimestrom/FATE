# MVP Scope

The MVP focuses on the protocol path that must exist before a richer trading interface can matter.

## Implemented

- Event signal ingestion
- Event candidate discovery
- AI market drafting and analysis
- YES / NO binary pool pricing
- Buy-side trade execution
- Sell-side position exit
- Account collateral and position ledger
- Protocol fee accounting
- Market close-time locking
- Mark-to-market portfolio valuation
- Market resolution with evidence
- Resolution evidence reliability checks
- User redemption after settlement
- Solana instruction builders
- Deterministic protocol tests
- Bundled Fate Market brand assets

## Next Modules

- Wallet session integration
- Persistent storage
- Historical market indexing
- Real-time event feeds
- AI provider adapters
- Market creator permissions
- Solana program client
- Frontend trading app

## Product Standard

Every future feature should preserve this loop:

```text
information -> market -> price -> trade -> evidence -> settlement
```
